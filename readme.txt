# Presidential Tweets Sentiment Analysis

This project analyzes sentiment in tweets related to presidential candidates using both traditional machine learning models and transformer-based deep learning approaches.

## Features

- Sentiment classification (positive/neutral/negative)
- Comprehensive text preprocessing:
  - Emoji handling
  - Negation handling
  - Lemmatization
  - Stopword removal
- Multiple model types:
  - Logistic Regression
  - SVM
  - Random Forest
  - Naive Bayes
  - DistilBERT
- Model interpretation with SHAP and BERTviz
- Bias analysis by political party
- Temporal sentiment analysis
